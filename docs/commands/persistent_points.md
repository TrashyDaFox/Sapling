---
title: Persistent Points
category: Utilities
---

Persistent points allow you to set points per entity that can be teleported to.

NOTE: Persistent points will be getting new features soon, some of these might not be fully implemented in current public versions of leaf before v4.1.3

## Setting / Teleporting

Setting a point: `/leaf:pers_point_set <entity> <position> <name>`
Teleporting to a point: `/leaf:pers_point_tp <entity> <name>`
Deleting a point: `/leaf:pers_point_del <entity> <name>`

## Global

Setting a point: `/leaf:g_pers_point_set <position> <name>`
Teleporting to a point: `/leaf:g_pers_point_tp <entity> <name>`
Deleting a point: `/leaf:g_pers_point_del <name>`
